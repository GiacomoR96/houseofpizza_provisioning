
# Base command for using terraform

terraform init — downloads plugins needed to execute config files (does note modify or apply state or config files)

terraform validate — validates if the file syntax is valid

terraform plan — creates an execution plan, listing resources that will be created based on the Terraform file

terraform apply — provisions/creates the resource infrastructure as defined in the Terraform file

terraform destroy — deletes/removes all resources that were created with the terraform apply command

terraform fmt — formats the spacing to correct HCL format
